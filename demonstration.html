Chatting mode

Running the Ollama llama 3.2:1b, Piper TTS, and OpenAI Faster Whisper, BMO can not only converse but also have multiple face animations and runs completely local.

Study mode

In study mode, BMO tracks user's focus by using the Yolo v11 model and Pi AI camera, and uses a scoring mechanism to tell the user how focused they were.

Gaming mode

In gaming mode, BMO boots up ES-DE and allow users to select through a variety of retro games using the built in d-pad and you can game with friends!

Music mode

In music mode, BMO can play from a selection of pre-selected songs.

Canvas feature

Along with the different modes, BMO also has a Canvas feature allowing users to fetch their assignment due dates and grades in chat mode via the CanvasAPI.
